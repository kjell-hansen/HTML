Förformatterad text

Förformatterad text är egentligen inte formatterad utan det handlar om att textens white space behålls i det skick som den är i källkoden. Oftast blir teckensnittet ett med fast bredd:

Denna texten finns inom pre-taggar
och behåller alltså   all    white space
som 
man 
skriver in i     koden.

Källkod

Om man någon gång behöver skriva in källkod på en sida gör man det lämpligen inom code-taggar:

function showHint(str) { if (str.length == 0) { document.getElementById("txtHint").innerHTML = ""; return; } else { var xmlhttp = new XMLHttpRequest(); xmlhttp.onreadystatechange = function() { if (this.readyState == 4 && this.status == 200) { document.getElementById("txtHint").innerHTML = this.responseText; } }; xmlhttp.open("GET", "gethint.php?q=" + str, true); xmlhttp.send(); } }

Där behålls inte white space och radbrytningar som i originaltexten men teckensnittet blir ett med fast bredd.
Via css kan man emellertid få texten att behålla white space och radbrytningar.
